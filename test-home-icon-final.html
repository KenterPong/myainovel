回首頁圖標功能測試 - 最終版本

✅ 功能實現完成：

在單一章節視圖中，導航區域會顯示四個按鈕：首頁、上一章、章節列表、下一章

功能特點：

  • 保持原有功能：章節列表圖標功能完全保留
  • 新增首頁按鈕：在上一章按鈕左邊添加獨立的首頁按鈕
  • 智能顯示：只在單一章節視圖中顯示首頁按鈕
  • 平台首頁：點擊首頁按鈕返回平台首頁（章節列表）

場景 1：首頁列表模式（showHomeButton = false）

只顯示三個按鈕：上一章、章節列表、下一章

場景 2：單一章節視圖模式（showHomeButton = true）

顯示四個按鈕：首頁、上一章、章節列表、下一章

按鈕功能說明：

  • 首頁按鈕（房子圖標）：返回平台首頁（章節列表）
  • 上一章按鈕（左箭頭）：導航到上一章節
  • 章節列表按鈕（三條橫線）：打開章節列表彈窗
  • 下一章按鈕（右箭頭）：導航到下一章節

實現細節：

  • ChapterNavigation 組件：添加了獨立的 HomeButton 組件
  • 保持原有功能：MiddleButton 始終顯示章節列表圖標
  • 條件顯示：首頁按鈕只在 showHomeButton = true 時顯示
  • 布局順序：首頁 → 上一章 → 章節列表 → 下一章

測試步驟：

  1. 訪問 http://localhost:3000
  2. 點擊某個章節進入單章節視圖
  3. 檢查導航區域是否顯示四個按鈕（首頁在最左側）
  4. 點擊首頁按鈕，確認返回章節列表
  5. 點擊章節列表按鈕，確認彈窗功能正常
